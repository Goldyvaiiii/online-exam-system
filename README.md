# Online Exam System

A full-stack, beginner-friendly web application where teachers can create and manage MCQ exams, and students can take those exams online with a live timer.

## Tech Stack
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Authentication**: JWT & Bcrypt (No TypeScript used)

## Features
- **Teachers**: Register/Login, create exams, add questions with 4 options, view student scores.
- **Students**: Register/Login, view available exams, take exams with a countdown timer, view results immediately.

## Project Structure
- `/backend`: Node.js & Express API, containing controllers, routes, middleware, and database connection.
- `/frontend`: Vanilla HTML, CSS, and JS frontend with separate dashboards for teachers and students.
- `schema.sql`: MySQL database schema defining the required tables and relationships.
